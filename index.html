// Standard Korean Keyboard (두벌식 / Dubeolsik)

      

    

    

      How to enable Korean on your device:

      PC (Windows): Settings → Time & Language → Language → Add Korean → then press Alt+Shift to switch

      iPhone: Settings → General → Keyboard → Keyboards → Add New Keyboard → Korean (2-Set) → tap 🌐 to switch
    

    

      두벌식 Rules:

      Consonants (자음) = left side of keyboard  |  Vowels (모음) = right side

      Syllable = Initial consonant + Vowel (+ optional final consonant)

      Example: 보안 (security) → ㅂ+ㅗ+ㅇ+ㅏ+ㄴ → type: q h d k s
    

    

      Timer color guide:

      🟢 Green = on pace  |  🟡 Yellow = a bit slow  |  🔴 Red = slow

      Pace is calculated per-character so longer phrases give you more time.
